# RunBang
跑步帮app

针对跑步运动，为年轻人定制的跑步社交APP

采用Material Design设计原则，使用Bmob后端云，百度地图SDK,ImageLoader等实现

应用官网： http://fir.im/runbang

注意：
 1、后端使用Bmob后端云，可能长时间未调用API，Bmob关闭应用,会导致出现无法登陆，无法注册、服务器错误等。可以邮件联系我解决。
 2、该应用只是以前做的Demo,已不在更新维护，仅供学习参考使用。
